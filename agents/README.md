# HerCircle AI - Backend ve Otomasyon Ajanları (Agents)

Bu klasör, HerCircle platformunun sunucu tarafındaki (backend) yapay zeka otomasyon süreçlerini yöneten akıllı ajanları içerir. 

Şu aşamada projemizin ön yüzündeki (frontend) yapay zeka etkileşimleri temel düzeyde tarayıcıda çalışsa da, sistemin gelecekteki mimarisi için Doğal Dil İşleme (NLP) destekli arka plan ajanları burada projelendirilmiştir.

## İçerik
- `support_agent.py`: Kullanıcının uygulamanın sohbet (chat) kısmından yazdığı metinleri analiz eden (sentiment analysis) ve kullanıcının psikolojik, hukuki veya eğitim ihtiyaç tasnifini otomatik olarak yapan bir Python otomasyon ajanı taslağıdır. Bu ajan gelecekte OpenAI veya açık kaynaklı dil modelleriyle tam entegre edilecektir.
