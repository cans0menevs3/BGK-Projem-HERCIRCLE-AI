# Kullanıcı Akışı (User Flow)

1. Kullanıcı uygulamayı açar  
2. Ana ekranda problem giriş alanını görür  
3. Problemini yazar veya sesli anlatır  
4. "Analiz Et" butonuna basar  

5. Yapay zeka:
   - Problemi analiz eder  
   - Kategori belirler  
   - Öncelik sıralaması yapar  

6. Kullanıcıya gösterilir:
   - Çözüm planı  
   - Yapılacak adımlar  

7. Sistem önerir:
   - Mentor / destek veren kullanıcılar  

8. Kullanıcı destek talep eder  

9. Görevler tamamlanır  
10. Kullanıcı ilerleme kaydeder
